## 题目分析
- 有6个嫌疑犯罪人 ， 分别给他们提了：
  + A，B两人中至少有一人参与作案。
  + A、D两人不可能是同案犯。
  + A、E、F三人中有两人参与此案。
  + B、C两人或者同时参与此案，或者都与此案无关。
  + C、D中有一人也只有一人系案犯。
  + 如果D没有参与次案，则E也没有不可能参与作案。
  
的 线索 。

根据以上信息 ， 可得出 ：
  + 代码需要6个变量
  + 用6个变量当循环变量来循环
  + 写完循环把上方条件用if（）来判断

注：0 = 无作案 ， 1 = 作案 。
## 代码如下：
```
#include <stdio.h>
#include <stdlib.h>

int main( int a , b , c , d , e , f)
{
	printf("1 = 作案 \n0 = 无作案 \n");
	for ( a = 0 ; a <= 1 ; a ++ )
	for ( b = 0 ; b <= 1 ; b ++ )
	for ( c = 0 ; c <= 1 ; c ++ )
	for ( d = 0 ; d <= 1 ; d ++ )
	for ( e = 0 ; e <= 1 ; e ++ )
	for ( f = 0 ; f <= 1 ; f ++ )
	{
		if ( a + b == 0 ) continue;
		if ( a + d == 2 ) continue;
		if ( a + e + f != 2) continue;
		if ( b != c ) continue;
		if ( c + d != 1 ) continue ;
		if ( d == 0 && e == 1 ) continue;
		
		printf("a=%d\nb=%d\nc=%d\nd=%d\ne=%d\nf=%d\n",a,b,c,d,e,f);
	}
	return 0 ;
}


```
